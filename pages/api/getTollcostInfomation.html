API介绍

API Introduction

收费信息服务API是一组支持收费信息查询、用户反馈等功能的服务接口，是收费信息服务产品的核心服务。内容包括：收费查询API、用户反馈API等

收费信息查询API：计算收费站之间指定路径的收费金额的服务接口。输入起始收费站和终止收费站之间的路径坐标，包含起始收费站和终止收费站的坐标，服务接口返回该路径的预估出行费用

用户反馈API：支持用户上报纠错信息，包含：路线，金额，车型等
